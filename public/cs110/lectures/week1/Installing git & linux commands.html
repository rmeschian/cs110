Introduction to Linux comands

most used

ls

This is the list command

Show all of the major directories

cd

This is the change directory command

Allow the user to change between file directories.

mv

This is the move command

Allows a user to move a file to another folder or directory

man

This is the manual command

Used to show manual of the inputted command

cp

This is the copy command

Used to copy files and directories

cat

cat command is used to display the content of text files and to combine several files to one file. The cat command does not accept directories

mkdir

This is the make directory

Allows the user to make a new directory

rmdir

This is the remove directory command

Allows the user to remove an existing directory or folder

rm

This is the remove command

The rm command will deleted created files

clear

Using the clear command will take the user back to the start prompt of whatever directory you are currently operating in
