Named values (variables)

Notes

  1. variable: using a name to refer to a value
  2. create a variable using 'const'
  3. put a value into it using '='

Variable naming rules

  • Contain letters, digits, underscores, and dollar signs.
  • Begin with a letter, _ or $ (not a number)
  • Case sensitive (y and Y are different variables)
  • Reserved words (like JavaScript keywords) cannot be used as names
