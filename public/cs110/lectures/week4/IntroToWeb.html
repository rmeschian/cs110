World Wide Web Technologies

Tim Berners-Lee

Father of the World Wide Web

Tim Berners-Lee

  1. 1980s Working at "European Organization for Nuclear Research" in Geniva
  2. Physicists from around the world needed to share data
  3. Data stored in a variety of formats, on incompatible computer systems
  4. Solution: A single unified documentation system
  5. Technologies: HTTP, HTTP Server, HTML, Browser

HyperText Transfer Protocol (HTTP)

  • Request-response protocol in the client-server computing model
  • HTTP is an application layer protocol
  • Transport protocol used is usually TCP (Transmission Control Protocol), but can also be configured to use UDP (User Datagram Protocol)
  • HTTP has actions
    • GET
    • POST
    • etc...

HTTP Server

  • Web server is a computer program + hardware that delivers (serves) content, such as web pages, to a requesting client
  • Responds to HTTP requests (GET/POST/etc)
  • Serves html pages, images, javascript files, etc...

HyperText Markup Language (HTML)

  • Describes how data should be organized when presented to the user
  • Consists of tags of the form: <tagname>content</tagname> and <tagname/>
  • Similar to XML (with looser rules...XHTML has the tight rules of XML)
  • File name extensions: .html, .htm

Web Browser

  • Browser does not display the HTML tags, but uses the tags to interpret the content of the page
  • First browsers were text-based only
  • Mosaic: first graphical browser
  • Netscape Navigator - productized Mosaic

Uniform Resource Locator (URL)

http://www.google.com:80/search?q=yerevan&sort=1

  • Protocol
  • Address
  • Port
  • Path
  • Parameters

HTML Example

<!DOCTYPE html>
<html>
    <head>
        <title>Sample page</title>
    </head>
    <body>
        <h1>Hello World</h1>
    </body>
</html>
  • DOCTYPE declaration defines the document type
  • Text between the 'html' tags describes the web page
  • Text inside the 'head' tags is not rendered, put initialization logic here
  • Title of the page
  • Text inside the 'body' tags is the visible page content
  • Header block element with the Hello World text inside

HTML Overview

  • Inline elements: <b>, <td>, <a>, <img>, <label>, <span>
  • Block elements: <h1>, <p>, <ul>, <ol>, <table>, <div>
  • General containers: <div> and <span>
  • Form elements: <form>, <button>, <input>, <textarea>, <select>
  • Comments in html: <!-- this is a comment -->
  • Use lowercase when specifying tags - as recommended by the W3C

CSS tips

  • CSS can be placed on elements using the 'style' attribute, but avoid doing this
  • CSS can be placed inside 'style' tags at the top, but avoid doing this
  • Put CSS in external files with .css extensions, reference them inside your 'head' tags: 
    <link rel="stylesheet" href="../css/main.css">
  • Order matters!
  • Most specific selectors win
